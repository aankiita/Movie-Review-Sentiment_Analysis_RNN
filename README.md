# Movie-Review-Sentiment_Analysis_RNN
This project is a movie review sentiment analysis web app using a Simple RNN neural network in TensorFlow/Keras. Users can input a movie review and the app will classify it as Positive or Negative.

FEATURES:
      Uses the IMDB dataset for training sentiment analysis.
      Implements a Simple RNN for sequence processing.
      Built with Streamlit for an interactive web interface.
      Real-time sentiment prediction for user input reviews.
      Displays both sentiment and prediction score.
